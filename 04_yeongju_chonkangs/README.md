# 🏡 빈집 활용 촌캉스 사업 — 2024 영주시 데이터 분석·활용 공모전

영주시의 빈집 문제와 청년 일자리·관광 침체 문제를 동시에 해결하기 위해, 지리 데이터를 기반으로
촌캉스 숙박 사업의 최적 입지를 도출한 프로젝트입니다. 3인 팀에서 팀장을 맡아 주제 선정부터
데이터 분석, 코드 구현, 발표까지 프로젝트 전 과정을 이끌었습니다.

## 📌 문제 정의
영주시는 인구 감소로 인한 빈집 증가 문제를 겪고 있었고, 동시에 청년 일자리 부족과 관광 산업
침체 문제도 안고 있었습니다. 마침 '촌캉스'가 새로운 여행 트렌드로 떠오르고, 농어촌 빈집을
숙박업으로 활용할 수 있는 규제 샌드박스 실증특례가 연장된 상황이었습니다. 이 세 가지 흐름을
연결해, 방치된 빈집을 촌캉스 숙박시설로 전환하는 사업을 제안했습니다.

## 🔍 핵심 제약과 해결 방법
빈집의 정확한 주소는 개인정보 문제로 제공받을 수 없었고, 읍면동 단위 등급별(1-4등급) 빈집
개수 데이터만 주어졌습니다. 이 제약을 해결하기 위해 **19개 읍면동 각각의 행정복지센터 위치를
해당 지역 빈집의 대표 좌표로 삼는 방식**을 직접 제안해 분석을 진행할 수 있는 방법을 찾아냈습니다.
숙박업 전환이 용이한 1-2등급 빈집만 분석 대상으로 삼고, 철거가 필요한 3-4등급은 제외했습니다.

## 📊 분석 흐름
1. **데이터 전처리**: 관광지·맛집·빈집·버스정류장·행정복지센터 데이터 정리 (숙박시설은 경쟁시설이므로 관광지 목록에서 제외)
2. **빈집 데이터 재분배**: 통계상 하나로 묶여 있던 동(가흥동/영주동/휴천동)을 실제 세부 행정동 면적 비율로 재분배
3. **지오코딩**: Google Maps Geocoding API로 주소를 위경도 좌표로 변환
4. **지도 시각화**: Folium으로 관광지·맛집·정류장·행정복지센터를 지도에 표시
5. **거리 계산**: geopy로 각 행정복지센터와 주변 시설 사이의 거리(km) 산출
6. **종합 점수화**:
   ```
   종합 점수 = 인기도 점수 + 주거환경(빈집 등급) 점수 − 평균 거리
   ```
   - 인기도 점수: 거리가 가까울수록, 관광지·맛집 순위가 높을수록 가중치 부여 (`1 / (거리+0.1) × 순위`)
   - 주거환경 점수: 1등급 빈집일수록 높은 가중치 부여 (1등급×4 ~ 4등급×1)

## 🏆 최종 결과
종합 점수 기준 상위 지역으로 **문수면, 순흥면, 부석면**을 최종 촌캉스 활용 후보지로 선정했습니다.

## 👤 본인의 역할
팀장으로서 주제 선정, 분석 방향 설정, 전체 코드 구현, 중간보고 및 최종 발표를 총괄했습니다.
특히 빈집의 정확한 주소를 받을 수 없는 상황에서 행정복지센터를 대리 좌표로 활용하자는 아이디어를
직접 제안해 분석을 진행할 수 있는 방법을 찾아냈습니다.

## 💡 배운 점
원하는 데이터(정확한 주소)를 받을 수 없는 제약 상황에서도, 대체 가능한 데이터(행정복지센터 위치)를
찾아 분석을 이어갈 수 있다는 것을 경험했습니다. 데이터 분석에서는 이상적인 데이터가 항상 주어지지
않으며, 제약 조건 안에서 대안을 찾는 것도 분석가의 역량이라는 점을 배웠습니다.
